# Rope Untangling 繩結出題工房／解謎

忠實採用實體盤面核心數量與自由端上下穿越邏輯的 2.5D 瀏覽器出題與遊戲工具。

## 盤面規則

- 22 個圓周插孔
- 10 條不同顏色的彈力繩（20 個端點）
- 完成出題後保留 2 個空洞
- 每條新繩可從既有繩索下方穿過最多 2 次
- 每條既有繩最多提供 3 個被動交匯節點
- 被動節點順序：中點 `1/2`、`1/4`、`3/4`
- 新繩與通過交匯點後的自由段都回到最上層
- 點擊既有繩只記錄一次「下穿」，不會立即建立繩結
- 放下終點後，才依起點與終點相對於目標繩的側別結算拓撲

### 下穿與扭轉判定

假設目標繩把盤面分成兩側，新繩起點位於其中一側：

| 下穿點擊次數 | 終點位置 | 結果 | 遊戲清除次數 |
|---:|---|---|---:|
| 1 | 與起點異側 | 單純下穿，沒有扭轉 | 1 |
| 1 | 與起點同側 | 單層扭轉 `×1` | 1 |
| 2 | 與起點異側 | 雙重扭轉 `×2` | 2 |
| 2 | 與起點同側 | 單層扭轉 `×1` | 1 |

側別使用交匯點處目標繩的局部切線計算，所以彎曲繩索也適用同一規則。單純下穿雖然不是扭轉，但該局部仍位於目標繩下方，因此會阻擋取繩，必須在遊戲中跨過目標一次才清除。

## 出題模式

1. 點擊空洞，選擇目前繩子的起點。
2. 可點擊已存在的繩索零至兩次，記錄自由端從其下方穿過。
3. 下穿後的自由段仍在最上層；重複點擊同一目標會預覽下穿、上方繞回、再次下穿的路徑。
4. 點擊另一個空洞放下終點，此時才結算為單純下穿、`×1` 或 `×2`。
5. 重複至十條繩完成，再按「開始遊戲」。

出題模式支援復原、清空、相同 seed 重設與隨機完整出題。隨機題在顯示前會先通過合法移孔搜尋；手動題也必須找到完整解才可進入遊戲。

## 遊戲模式

1. 點擊任一彩色繩端。
2. 點擊一個發光空洞，將端點換到該洞；原洞會立即變成空洞。
3. 移動軌跡只會清除該繩既有的下穿／扭轉，不會創造新結。
4. 單純下穿和 `×1` 都需越過目標一次；`×2` 第一次降為 `×1`，第二次才完全清除。
5. 自身沒有未清除層、且沒有較晚放置繩索壓在上方的繩會發光。
6. 點擊發光繩身取下；依出題逆序取下全部十條繩即完成遊戲。

遊戲模式提供剩餘繩索、移孔次數、未清除層數、重新開始與返回出題；桌機和手機會自動調整盤面與觸控尺寸。
