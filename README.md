# Rope Untangling 繩結出題工房／解謎

忠實採用實體盤面核心數量的 2.5D 瀏覽器出題與遊戲工具。

## 盤面規則

- 22 個圓周插孔
- 10 條不同顏色的彈力繩（20 個端點）
- 完成出題後保留 2 個空洞
- 每條繩主動纏繞最多 2 次
- 每條繩被動纏繞最多 3 個節點
- 被動節點順序：中點 `1/2`、`1/4`、`3/4`
- 同一對繩的第二次纏繞會在同節點形成雙圈螺旋
- 新繩預設在最上層；交錯點局部換層，通過後回到最上層

## 出題模式

1. 點擊空洞，選擇目前繩子的起點。
2. 可點擊已存在的繩索零至兩次，加入纏繞。
3. 點擊另一個空洞完成該繩。
4. 重複至十條繩完成。
5. 按下「開始遊戲」進入簡化解謎畫面。

出題模式支援復原、清空、相同 seed 重設與隨機完整出題。隨機題在顯示前會先通過合法移孔搜尋；手動題也必須找到完整解才可進入遊戲。

## 遊戲模式

1. 點擊任一彩色繩端。
2. 點擊一個發光空洞，將端點換到該洞；原洞會立即變成空洞。
3. 移動軌跡只會解除該繩既有的纏繞，不會創造新結。
4. 每次越過一條已纏住的目標繩，只解除一圈；`×2` 必須越過兩次才完全解開。
5. 自身沒有未解纏繞、且沒有較晚放置繩索壓在上方的繩會發光。
6. 點擊發光繩身取下；依出題逆序取下全部十條繩即完成遊戲。

遊戲模式提供剩餘繩索、移孔次數、未解圈數、重新開始與返回出題；桌機和手機會自動調整盤面與觸控尺寸。

